WinterWood

In this school project, I collaborated with two classmates as part of the Fullstack Development course. The goal was to apply our web design skills and knowledge of JavaScript in a real-world context by building a fully functional website for a Bed & Breakfast.

As part of the project, we developed an API using Python to fetch and manage data stored in a hosted SQL database. This data was then dynamically displayed on the website using JavaScript, allowing us to create a responsive and data-driven user experience.

What I Learned

During this project, I significantly expanded my technical skillset. I learned how to build an API using Python and how to interact with APIs on the front-end using JavaScript. I also gained hands-on experience with SASS for styling, and improved my understanding of GitHub workflows, including working with branches and managing version control collaboratively.

One of the most valuable takeaways was learning how to host APIs and SQL databases using Microsoft Azure, as well as how to deploy the front-end using Netlify. We also worked with API keys to manage secure access between the front-end and back-end services. Above all, this project emphasized the importance of teamwork, efficient communication, and collaboration throughout the development process.
